FixItNow – Neighborhood Service & Repair Marketplace 🛠️

FixItNow is a full-stack web application that connects residents with nearby service professionals such as electricians, plumbers, carpenters, and appliance repair experts. Customers can post service requests, chat with providers, and leave reviews — while providers can manage bookings and profiles. Admins oversee the platform for quality and security.

✨ Features:

👨‍💼 Customer

Register/Login with secure authentication

Browse and request services from nearby providers

Track booking status (Requested → Accepted → Completed)

Chat with service providers

Rate and review completed services

🧑‍🔧 Service Provider

Manage profile and offered services

View and accept job requests

Update service request status

Receive ratings and feedback

🛡️ Admin

Manage users and providers

Approve new providers

Monitor service activity and feedback

🧩 Tech Stack Layer Technology Frontend React.js, HTML5, CSS3 Backend Spring Boot, REST APIs Security JWT Authentication Database MySQL + JPA/Hibernate Build Tools Maven (Backend), NPM (Frontend)
